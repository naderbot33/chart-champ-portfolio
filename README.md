# Chart Champ Portfolio

Static public portfolio dashboard for the Chart Champ community.

## Files

- `index.html` - page structure and required dashboard sections.
- `assets/styles.css` - responsive dashboard styling.
- `assets/app.js` - calculations, ticker search, scheduled snapshot loading, and rendering logic.
- `data/portfolio-data.js` - starter portfolio data plus sourced static fallback prices.
- `data/live-prices.json` and `data/live-prices.js` - public quote snapshots generated by GitHub Actions.
- `scripts/update-portfolio-quotes.mjs` - refreshes the public quote snapshots from Yahoo Finance chart data.

## Quote Refresh

The portfolio quote snapshot is refreshed by GitHub Actions using Yahoo Finance
chart data. The page also shows a quote freshness note so stale snapshots are
visible instead of silently looking current.

Run manually:

```sh
node scripts/update-portfolio-quotes.mjs
```
